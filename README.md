# Qwen Local Chat
原生 Qwen1.0 FP16 大模型本地离线部署 | 16G 显存极限优化 | CPU+GPU 异构推理    
基于 Qwen 大模型的纯本地离线 AI 对话助手，无需联网、无需 API Key、自带多主题对话记忆，跨设备稳定运行。
制作这个项目原本是因为不想把对话数据传到云端，希望有一个完全属于自己电脑、不泄露隐私、随时可用的本地 AI。

## 功能特性
- 纯本地离线运行，不联网、不上传数据、不泄露隐私
- 完整多主题对话管理，不同话题互不干扰，记忆完全隔离
- 对话历史自动本地保存，退出不丢失，下次打开继续聊
- AI 流式逐字输出，阅读体验更流畅
- CPU / GPU 自动识别适配，不挑设备
- 命令行简洁交互，常用操作一键完成
- 无广告、无内购、无付费解锁
- 完整工程化结构，可二次开发

## 项目结构
- conversation_history/    # 对话历史自动保存目录
- models/                  # 本地 Qwen 模型存放目录
- main.py                  # 主程序
- cuda_ok.py               # CUDA 环境检测工具
- requirements.txt         # 依赖清单
- README.md                # 说明文档

## 运行说明
本项目为 原生 FP16 模型部署，非量化版本，对显存要求较高。在 16G 显存环境下，模型会自动进行显存 / 内存 / CPU 调度，实现稳定推理。（作者运行环境内存32GB！）

## 环境要求
- Python 3.12（必须，3.13 会报错，更低版本不保证兼容）
- NVIDIA GPU + CUDA（可选，但强烈推荐，速度提升巨大，作者用的是cuda128）
- 已下载好的 Qwen 本地模型文件

## 使用前安装依赖
```bash
pip install -r requirements.txt
```
- 注意！这里的 requirements.txt 是基于作者的环境生成的！作者环境为 rtx5070ti。50系卡可以无视，40系及以下请修改torch版本或直接执行以下指令并删除 requirements.txt 中的这三个模块!
```bash
pip install torch torchvision torchaudio
```
- 并且如果您为50系的卡且执行了第一条指令必会报错！因为只有测试版torch可以上50系的卡，需要使用这句指令先安装torch测试版（cuda128）再执行第一句指令！
```bash
pip install --pre torch torchvision torchaudio --index-url https://download.pytorch.org/whl/nightly/cu128
```

## 检查 CUDA 是否可用
```bash
python cuda_ok.py
```

## 启动助手
```bash
python main.py
```

## 内置命令
- /help 查看帮助
- /topics 查看所有对话主题
- /create 名称 创建新主题
- /switch 名称 切换主题
- /delete 名称 删除主题
- /history 查看当前对话记录
- /clear 清空屏幕
- /quit 保存并退出

## 模型放置
- 请将下载好的 Qwen 模型放入 **./models** 文件夹

## 安全与隐私说明
- 所有对话仅保存在你自己电脑，不上传任何服务器
- 模型完全本地运行，无需联网，无网络追踪
- 历史记录以 JSON 格式存储，可随时手动删除
- 不收集设备信息、不收集对话内容

## 关于性能
- 有 NVIDIA 独显：秒回，体验接近在线大模型
- 无独显 CPU 运行：可以用，但速度较慢
- 建议显存 ≥ 6GB（7B 模型）

## 写在最后
这个项目原本是想搞一个本地可用的AI助手，没有付费并保护隐私。但考虑到是非量化模型，普通用户使用应该有点难。不过未来也许会支持qwen3量化也不一定（有时间吧。。。）。总之，这个东西不太完美，但是能用，稳定。
